Programming Models

This project report on Indian Flight Pricing analyzes factors influencing ticket prices and develops predictive models. The study explores a dataset from six metro cities in India over 50 days, implementing regression models such as Linear Regression, Decision Tree, and Random Forest. The results highlight the importance of seat class, flight duration, and days left until the flight in determining ticket prices, providing valuable insights for airlines and individuals in pricing strategies and travel planning.

This project report analyzes the Residential Apartment Price Factor Data from Equity Residential, focusing on predicting the monthly rent price based on various apartment characteristics. Three regression models, namely K-Nearest Neighbor Regression, Linear Regression, and Regression Tree, are employed to determine the best model for price prediction. The significant variables affecting the price are identified and their impact on the apartment price is examined. Ultimately, this model can be used to find good value when looking for apartments. Finding apartments with pricing lower than what the model suggests implies a good deal!

SQL and Database

The report covers the organization of data for tracking individual athletes' performances in various sports events. It includes an Entity-Relationship Diagram (ERD) to visualize the database structure. The report also provides implementation code for creating and populating tables such as Athlete, Sport, Competition, Heat, Performance, DistanceResult, and TimeResult. Additionally, it presents SQL commands to retrieve specific information from the database, such as listing event details on a specific date, counting the number of events for men and women, finding the longest distance in the Long Jump Women's event, displaying runner names, times, and rankings in the 100M event, determining the count of athletes participating in multiple events, and identifying the longest distance in the Long Jump Women's event.
